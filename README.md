# jimin-kim-2
# 오픈마켓 웹 서비스 (개인 프로젝트)

## 프로젝트 소개
HTML, CSS, JavaScript를 활용하여 구현한 오픈마켓 웹 서비스입니다.  
회원가입 / 로그인부터 상품 목록, 상품 상세 페이지까지의 기본적인 쇼핑몰 흐름을 구현했습니다.

팀 프로젝트로 시작했으나, 개인 프로젝트로 전환하여 전체 기능을 단독으로 구현했습니다.

---

## 사용 기술
- HTML5
- CSS3
- JavaScript (ES6)
- Git / GitHub
- LocalStorage

---

## 구현 기능

### 회원 기능
- 회원가입 (구매회원)
- 로그인 / 로그아웃
- 로그인 상태 유지 (LocalStorage)

### 상품 기능
- 상품 목록 페이지
- 상품 카드 클릭 시 상세 페이지 이동
- 상품 상세 정보 조회
- 수량 선택 및 총 금액 계산
